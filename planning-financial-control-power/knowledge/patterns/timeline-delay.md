# Pattern: Timeline Delay

## Trigger

Milestone date changes or dependency delay appears.

## Affected graph path

```txt
MS -> RA -> COST -> FCST -> RPT
```

## Required skills

```txt
timeline-planning
resource-planning-allocation
cost-calculator
report-builder
fact-check
```

## Required output

```txt
timeline impact
resource extension/conflict
cost delta
forecast variance
RAG/report impact
decision needed
```

## Guardrail

Do not mark delay as confirmed unless milestone change has evidence, decision, or explicit assumption.
