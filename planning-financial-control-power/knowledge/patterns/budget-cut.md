# Pattern: Budget Cut

## Trigger

Approved budget or funding cap is reduced.

## Affected graph path

```txt
Budget -> FCST -> COST -> RA -> MS -> RPT
```

## Required skills

```txt
cost-calculator
resource-planning-allocation
timeline-planning
report-builder
fact-check
```

## Required output

```txt
new variance
resource reduction option
timeline/scope impact
risk created
decision options
```

## Guardrail

Do not propose reducing resources without checking milestone feasibility.
