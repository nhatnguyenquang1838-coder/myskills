# Use Case References

This folder stores stable reference material for Planning & Financial Control Power use cases.

## Purpose

Use these references when designing, testing, or improving agents/skills/hooks for the Power.

```txt
Use cases = what job the Power performs
Readiness modes = how much confidence/output authority is allowed
Control flows = how agents, skills, hooks, guardrails, and circuit breakers execute each use case
```

## Reference files

| File | Purpose |
|---|---|
| `critical-flows-with-controls.md` | Mermaid flows for simplest and most complex cases with agents/skills/hooks/breakers |
| `uc-control-summary.md` | Compact mapping from UC families to agents, skills, hooks, guardrails, bias, hallucination controls |
