# Timeline Planning Method

## Purpose

Convert scope/work packages into milestone and dependency nodes.

## Method

```txt
1. Identify work packages.
2. Define milestone per deliverable or phase.
3. Link dependencies.
4. Identify fixed dates.
5. Mark assumptions and missing data.
6. Determine cascade to resource/cost/report.
```

## Required graph links

```txt
WP -> MS -> DEP
MS -> RA
MS -> RPT
```

## Anti-pattern

Creating timeline dates without source, assumption, or decision IDs.
